Space Invaders

Description

This was my final project for my first computer science class 1133. I used Python and the TKinter library to recreate a little portion of the old Space Invaders game. Due to the time constraint of a one week project, only one level was remade. This is one of my favorite projects for a computer science class. I went back at one point to fefactor this and make it better, to use that time learning more C++ and graphics.

Code

Go to Github page

Tools Used

  • TKinter for the windowing and graphics

Features

  • Moveable player
  • Textured players and enemies
  • Shootable weapons for both players and enemies
  • Simple AI for enemy group movement
  • Co-op mode with a 2nd player ship
  • Highscore system and saving
  • Some collision detection between the bullet / enemies, and enemies / players
